// OBJECTIVES

โปรเจกต์นี้ใช้ AI + Automation Pipeline เพื่อช่วยทีม 8–12 คน ตัดสินใจเลือกร้านอาหารในย่านอารีย์ โดย scrape ข้อมูลจริงจาก Google Maps และ TripAdvisor แล้วทำ cleaning, scoring และ recommendation แบบอัตโนมัติ ลดเวลาการตัดสินใจ จาก "ไปกินที่ไหนดี" สู่ข้อมูลที่มีคะแนนและเหตุผลชัดเจน

01
Scrape ข้อมูลร้านอาหารจริงด้วย Apify Google Maps & TripAdvisor Scraper
02
Clean & deduplicate data ด้วย Claude AI — ลบ null, ซ้ำ, เติม N/A
03
Scoring model 100 คะแนน 6 มิติ เพื่อ rank ร้านอย่างเป็นระบบและโปร่งใส
04
ระบุ Top 3 ร้านพร้อมเหตุผลและ trade-off สำหรับกลุ่มทีม 8–12 คน
05
Export Excel 4 sheets + HTML Dashboard สำหรับ present ผลลัพธ์
06
ทดสอบ n8n Automation pipeline เพื่อ run workflow แบบ end-to-end
